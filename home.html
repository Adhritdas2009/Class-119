SKETCH IDENTIFICATION APPLICATION




Sketch Identification is an application in which you can draw a doodle, and the computer will try to match it with the pretrained model called 'DoodleNet' and show the confidence value and name of the doodle drawn on the canvas.


DoodleNet is a pretrained model which has sketches classified into 345 categories. Each category has been trained with lots of images. It normally gives accurate results most of the times.
